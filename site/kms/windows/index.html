Windows Activation¶

Usage¶

Start a Command Prompt as an Administrator.

slmgr /ipk <GVLK>
slmgr /skms kms.s10e.net
slmgr /ato

Note

You are supposed to replace <GVLK> with your desired edition and product KMS Client Setup Key.
